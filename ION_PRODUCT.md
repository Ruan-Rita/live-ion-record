## 🛠️ Stack Técnico Simples

### Frontend (Plugin/Extension):

```javascript
// Gravação
navigator.mediaDevices.getDisplayMedia()
  → MediaRecorder
  → Blob (WebM)
  → Download direto!

// Sem backend necessário pro Free! 🎉
```

---

### Backend (só Pro):

**Stack:**
- Node.js + Express
- Upload: Cloudflare R2 (barato)
- DB: PostgreSQL (Supabase free tier)
- Auth: Supabase Auth
- Deploy: Fly.io/Railway

**Custo inicial:** R$0-50/mês
